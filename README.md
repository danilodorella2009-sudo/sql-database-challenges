# 🗄️ Java & Databases: Portfólio de Modelagem Relacional

Este repositório foi desenvolvido com o objetivo de consolidar conceitos fundamentais de modelagem de dados relacionais e sua respectiva integração com aplicações Java. Ele funciona como um portfólio acadêmico, reunindo diferentes cenários de negócios, regras de integridade e estruturas de tabelas.
